# # Bank Account Management - Dart

This Dart program simulates a simple bank account system using Object-Oriented Programming (OOP).  
It demonstrates:

- Creating bank accounts with initial balance or zero balance
- Depositing money into accounts
- Withdrawing money from accounts
- Displaying account information

## Features

- Uses classes and constructors (including named constructor)
- Handles deposits and withdrawals with proper checks
- Displays clear messages for each transaction
- Demonstrates basic OOP principles in Dart

## How to Run

1. Open the Dart file (e.g., `bank_account.dart`) in VS Code
2. Run the program
3. Observe the deposits, withdrawals, and account information printed in the console
